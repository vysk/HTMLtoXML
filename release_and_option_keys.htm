Ordering and Entering Release and Option Keys

After the ova has been deployed as a Guest on the VM Host, and you have set the root and admin passwords, you can access the through a web browser and order your release key.

Order the Keys

  1. Log in to the via a web browser as admin.

  2. Follow the service setup wizard to define the purpose of the system, apply the release and options keys, then restart the system.

    The wizard helps you through the configuration described in the rest of this topic, so you can ignore the rest of the topic if you are using the wizard.

    If you do not want the wizard to guide you, click Skip Service Setup Wizard, then follow the rest of these instructions.

  3. Get release and option keys:
    1. Go to the Option keys page (Maintenance > Option keys).
    2. Copy the Serial number.

    3. Use this serial number to order release and option keys for this VM .

      For full details on obtaining your release and option keys, see Appendix 2: VM New Product Hold (NPH) release process [p.1].

Apply the Keys to the System

When you have the release and option keys:

  1. Log in to the via a web browser as admin.
  2. Enter the release and option keys:
    1. Go to the Option keys page (Maintenance > Option keys).
    2. Enter the release key provided in the Release key field.
    3. Click Set release key.
    4. For each option key provided, enter the key value in the Add option key field and click Add option.
  3. Reboot the to activate the licenses:
    1. Go to the Restart options page (Maintenance > Restart options).
    2. Click Reboot.

  4. After the reboot, log in to the web interface and configure the , including DNS, NTP, zones, search rules and so on as required.
